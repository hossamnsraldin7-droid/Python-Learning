01. ما هي البرمجة ولغة Python؟

| ما هي البرمجة؟

البرمجة ببساطة هي إعطاء تعليمات مرتبة للكمبيوتر ليقوم بمهمة معينة. تماماً كما تعطي وصفة طبخ لشخص ما، أنت تكتب خطوات مرتبة والكمبيوتر ينفذها بدقة.

أنت (المبرمج) تفرز وتكتب الكود ◀ الكمبيوتر يفسر وينفذ الكود ◀ النتيجة تظهر للمستخدم (مثال: احسب 5 + 3 ◀ يطبع: 8)

| ما هي لغة Python ومميزاتها؟

بايثون هي لغة برمجة عالية المستوى أنشأها المبرمج Guido van Rossum عام 1991. واكتسبت شهرة عالمية للأسباب التالية:

الميزة التفسير والتفصيل
بسيطة وسهلة قواعدها تشبه اللغة الإنجليزية العادية، وهي سهلة القراءة والكتابة مقارنة باللغات الأخرى.
متعددة الاستخدامات تستخدم في بناء المواقع الإلكترونية، تطبيقات الذكاء الاصطناعي، أتمتة المهام، وتحليل البيانات.
مكتبات ضخمة مجانية تحتوي على آلاف الأدوات والأكواد الجاهزة التي توفر عليك كتابة البرامج من الصفر.
الأكثر طلباً تعد حالياً من أعلى اللغات طلباً في سوق العمل العالمي ومجالات التكنولوجيا الحديثة.
💡 ملحوظة هامة: بايثون اليوم هي اللغة الأولى عالمياً في مجالات الذكاء الاصطناعي (AI) وتعليم الآلة (Machine Learning)، وبتعلمها تفتح لنفسك باباً مستقبلياً ضخماً جداً.
⚠️ أخطاء شائعة للمبتدئين:
  • الاعتقاد بأن البرمجة صعبة جداً من البداية؛ الحقيقة أنها مهارة تراكمية تحتاج فقط إلى الصبر والممارسة اليومية.
  • محاولة حفظ الأكواد دفعة واحدة بدلاً من فهم منطق وكيفية عمل الكود.
🧠 تمرين : فكر في 3 مهمات يومية تقوم بها على الحاسوب ويمكن أتمامها ببرنامج بايثون (مثال: حساب فاتورة محل، ترتيب أسماء ملفات، أو إرسال رسالة تذكير تلقائية).

03. المتغيرات وأنواع البيانات

المتغير (Variable): عبارة عن صندوق تخزين محجوز في ذاكرة الكمبيوتر العشوائية (RAM)، تعطيه اسماً معيناً وقيمة، لتتمكن من استدعاء هذه القيمة أو تعديلها لاحقاً في الكود.

# إنشاء متغيرات برمجية مختلفة وتخزين القيم داخلها
age = 20                 # متغير رقمي صحيح
name = "hossam"          # متغير نصي
height = 1.7897          # متغير رقمي عشري
is_student = True        # متغير منطقي (بوليني)

print(name)              # hossam : يطبع
print(age)               # يطبع: 20

| أنواع البيانات الأساسية (Data Types):

النوع بالإنجليزية الرمز في بايثون الاستخدام مثال تطبيقي
Integer int الأعداد الصحيحة (موجبة أو سالبة) بدون كسور مثل الأعمار والكميات. age = 20
Float float الأعداد العشرية والكسرية مثل الأسعار، المعدلات، والأطوال. gpa = 3.75
String str النصوص والرسائل والكلمات، وتوضع دائماً بين علامتي اقتباس. "name = "Ali
Boolean bool القيم المنطقية ولها خياران فقط إما صواب أو خطأ. is_active = True

| التحويل بين أنواع البيانات (Type Casting):

# التحويل من نص يحتوي رقم إلى رقم حقيقي للقيام بالعمليات الحسابية
str_num = "42"
num_int = int(str_num)     # يتحول إلى عدد صحيح: 42
num_float = float(str_num) # يتحول إلى عدد عشري: 42.0
back_to_str = str(100)     # يتحول العدد 100 إلى نص: "100"

# لمعرفة نوع أي متغير الحالي نستخدم الدالة type()
print(type(num_int))       # المخرجات: <class 'int'>
⚠️ قواعد تسمية المتغيرات (أخطاء شائعة):
  • لا يجوز وضع مسافة في اسم المتغير (خطأ: "my name = "Ali)؛ الصواب استخدام الشرطة السفلية "my_name = "Ali, و يمكن وضع الشرطة السفلية في وسط او في اخر اسم المتغير.
  • لا يجوز بدء اسم المتغير برقم (خطأ: "1name = "Ali).
  • القيم المنطقية يجب أن تبدأ بحرف كبير دائماً: True أو False وليس بأحرف صغيرة.

04. العمليات الحسابية والمنطقية

تتعامل لغة بايثون مع الأرقام والمعادلات الرياضية والمنطقية بكفاءة الآلة الحاسبة وأكثر، وإليك تفاصيل رموز هذه العمليات الحسابية:

العملية الرياضية الرمز في بايثون مثال النتيجة المتوقعة
الجمع + 5 + 3 8
الطرح - 4 - 10 6
الضرب * 3 * 4 12
القسمة العادية / 3 / 10 3.3333333333333335 (تعطي float دائماً)
القسمة الصحيحة // 3 // 10 3 (تتجاهل ما بعد الفاصلة تماماً)
باقي القسمة % 3 % 10 1 (مفيدة جداً للتحقق من الأعداد الزوجية والفردية)
الأس / القوة ** 3 ** 2 8 (تعني 2 أس 3)

| عمليات المقارنة والمنطق (Comparison & Logical Operators):

# عمليات المقارنة (تنتج دائماً True أو False)
print(5 == 5)   # True (هل يساوي؟)
print(5 != 3)   # True (هل لا يساوي؟)
print(10 > 20)  # False (أكبر من)

# العمليات المنطقية الربط والجمع
x = 10
print(x > 5 and x < 20)  # True (يجب تحقق الشرطين معاً)
print(x == 5 or x == 10) # True (يكفي تحقق شرط واحد )
print(not(x > 5))        # False (عكس النتيجة المنطقية الحالية)

05. إدخال وإخراج البيانات (input & print)

التواصل مع المستخدم هو أساس أي برنامج تفاعلي، حيث يستقبل البرنامج مدخلات من المستخدم، ثم يعالجها وفقاً للأوامر التي تمت برمجتها، ويعرض النتائج المناسبة له :

1. دالة الإدخال input()

تستقبل البيانات المدخلة من لوحة المفاتيح وتعيدها دائماً كنوع نصي (String). إذا كنت تريد أرقاماً او نوعا آخر من البيانات ، يجب تحويلها يدوياً:

name = input(" أدخل اسمك : ")
print("أهلاً بك يا " + name)

# استقبال رقم واستخدامه مباشرة ك رقم. او نوع آخر من البيانات
age = int (input("كم عمرك الحالي؟ "))
print("عمرك بعد عشر سنوات سيكون:", age + 10)

# وهكذا مع باقي انواع البيانات الاخرى

2. دالة الإخراج المتقدمة f-string

تعتبر f-string هي الطريقة الأسهل والأحدث لدمج المتغيرات داخل النصوص البرمجية دون الحاجة لاستخدام علامات الجمع المتكررة والمربكة :

user_name = "محمود"
user_age = 25

# لاستخدام دالة الطباعة المتقدمة . قم بوضع حرف إف قبل علامتي الاقتباس . واكتب النص مباشرة و عند استخدام متغير قم بوضعه داخل زوج من الاقواس المجعدة
مثال 👇
print(f"مرحباً، اسمي {user_name} وعمري الحالي هو {user_age} سنة.")

06. الجمل الشرطية (if, elif, else)

تسمح الجمل الشرطية للبرنامج باتخاذ قرارات مختلفة وتغيير مسار تنفيذ الأوامر بناءً على تحقق الشروط المحددة أو عدم تحققها :

grade = int(input("أدخل درجتك في الاختبار (من 100): "))

if grade >= 90 :
    print("ممتاز! 🎉")
elif grade >= 80 :
    print("جيد جداً 👍")
elif grade >= 60 :
    print("ناجح ومقبول ✔️")
else :
    print("راسب، حظاً أوفر في المرة القادمة ")
💡 قاعدة المسافات البادئة (Indentation): في لغة بايثون، لا تُستخدم المسافات البادئة لأغراض جمالية أو لتحسين شكل الكود فقط، بل تُعد جزءاً أساسياً من بناء البرنامج. فهي تحدد للغة بداية ونهاية الكتل البرمجية (Code Blocks)، وتوضح أي الأوامر تنتمي إلى نفس المجموعة وأيها مستقل عنها. وتُستخدم المسافات البادئة في الجمل الشرطية والحلقات التكرارية والدوال والفئات (Classes) وغيرها من التراكيب البرمجية. لذلك فإن كتابة المسافات البادئة بشكل غير صحيح قد يؤدي إلى ظهور أخطاء تمنع البرنامج من العمل أو تتسبب في تنفيذ الأوامر بطريقة غير متوقعة.

07. الحلقات التكرارية (for & while)

تستخدم لتكرار تنفيذ كود معين عدة مرات دون الحاجة لإعادة كتابته يدوياً، مما يوفر الوقت والجهد :

1. حلقة for (التكرار المحدد):

تستخدم عندما تكون على علم مسبق بعدد مرات التكرار المطلوبة، أو عند المرور على عناصر قائمة ما :

# syntax : for (variable) in range (start, stop)
نقطة التوقف = النقطة المراد التوقف عندها +1
طباعة الأرقام من 1 إلى 5 باستخدام 
range(1, 6)
for i in range(1, 6):
    print(i)

# المرور على عناصر قائمة نصية
fruits = ["تفاح", "موز", "مانجو"]
for fruit in fruits:
    print(f"أنا أحب فاكهة: {fruit}")

2. حلقة while (التكرار المشروط):

تستمر في التكرار طالما أن الشرط المكتوب بجانبها صحيح ولم يكسر :

# syntax : while condition : 
                الاوامر المراد تكرارها 
    count = 5
while count > 0 :
    print(f"العد التنازلي: {count}")
    count -= 1 
    # إنقاص قيمة المتغير في كل مرة لكي يتم كسر الشرط 
    و حتى لا ندخل في حلقة تكرار لانهائية والجهاز يولع
print("انطلاق! 🚀")

| التحكم في الحلقات باستخدام break و continue:

  • break: إيقاف الحلقة التكرارية تماماً والخروج منها فوراً قبل اكتمالها.
  • continue: تخطي الدورة الحالية فقط والانتقال مباشرة إلى الدورة التالية في الحلقة.

08. القوائم (Lists) وعمليات التقطيع

القائمة (List): هي هيكل بيانات مرن ومُرتب يسمح بتخزين مجموعة من العناصر (نصوص، أرقام، إلخ) في متغير واحد، وتبدأ الفهرسة فيها من الترتيب صفر .اي ان اول عنصر يكون ترتيبه صفر :

التكوين : اسم القائمة = [عناصر من نفس نوع البيانات مفصولة عن بعضها ب فاصلة]

my_list = ["خبز", "حليب", "بيض"]
print(my_list[0])  # يطبع العنصر الأول: خبز
print(my_list[-1]) # الفهرس السالب يعيد من النهاية: بيض

# العمليات الأساسية والتعديل على القوائم
my_list.append("جبن")        # إضافة عنصر في نهاية القائمة
my_list.insert(1, "زبدة")     # إضافة عنصر في موقع محدد (الفهرس 1)
my_list.remove("حليب")       # حذف عنصر معين عن طريق الاسم
my_list.sort()               # ترتيب القائمة أبجدياً أو تصاعدياً
print(len(my_list))          # معرفة عدد عناصر القائمة

| ميزة تقطيع الأجزاء (Slicing):

nums = [0, 1, 2, 3, 4, 5]
print(nums[1:4]) # يستخرج من الفهرس 1 إلى قبل 4: [1, 2, 3]
print(nums[:3])  # يستخرج من البداية إلى قبل 3: [0, 1, 2]
print(nums[::2]) # تخطي خطوتين (طباعة الأعداد الزوجية): [0, 2, 4]

09. القواميس (Dictionaries)

القاموس (Dictionary): هيكل بيانات يخزن المعلومات على شكل ثنائيات تتكون من (مفتاح : قيمة) أو ما يعرف بـ Key : Value، ويشبه جدول بيانات صغير بحيث المفاتيح تكون فريدة دائماً.

التكوين : اسم القاموس = {المفتاح1 : القيمة1 , المفتاح2 : القيمة2}
       لا يشترط ان يكون المفتاح و القيمة من نفس نوع البيانات 
             و لا يشترط ان تكون كل القيم من نوع بيانات واحد 
    person = {
    "name": "mohammed",      # نص
    "age": 20,               # عدد صحيح
    "height": 1.75,          # عدد عشري
    "student": True          # قيمة منطقية
}

# الوصول للقيم عبر المفتاح
print(person["name"])        # المخرجات: محمد
print(person.get("email"))   # محاولة الوصول الى القيمة بأمان في حالة ان المفتاح قد يكون غير موجود بدلا من التسبب في حوث خطأ  

# التعديل والإضافة والحذف
person["age"] = 21           # تعديل قيمة موجودة
person["email"] = "m@m.com"  # إضافة مفتاح وقيمة جديدة
del person["height"]         # حذف مفتاح بالكامل

# المرور على القاموس باستخدام اللوب
for key, value in person.items() :
    print(f"المفتاح: {key} -> القيمة: {value}")

10. الدوال (Functions)

الدالة (Function): هي عبارة عن كتلة برمجية منظمة من الأكواد يتم تجميعها وإعطائها اسماً معيناً للقيام بمهمة محددة، وتستخدم لإعادة استخدام الأكواد . منع التكرار . جعل البرنامج أكثر تنظيماً وسهولة في الصيانة . تقسيم المشكلة الكبيرة إلى أجزاء صغيرة.

# (Parameter) تعريف دالة بسيطة تستقبل وسيطاً 
def greet_user(name):
    print(f"مرحباً بك يا {name}! أتمنى لك يوماً سعيداً ")

# استدعاء الدالة 
greet_user("محمد")   # => مرحباً بك يا محمد! اتمنى لك يوماً سعيداً  
greet_user("سارة")   # => مرحباً بك يا سارة! اتمنى لك يوماً سعيداً  

# return إنشاء دالة تقوم بحساب عملية وإرجاع النتيجة باستخدام 
def add_numbers(a, b):
    return a + b

result = add_numbers(10, 5)
print(f"مجموع الأعداد هو: {result}") # => 15

def greet(name) :  # name -> Parameter
    print(name)

greet("Hossam")    # "Hossam" -> Argument

 # افتراضي Parameter استخدام
def greet(name="ahmed")   # تمت اضافة قيمة افتراضية و هي احمد 
   print( name )

greet()        # في حالة عدم إرسال قيمة سيطبع أحمد بدلاً من اعطاء خطأ
greet("Ali")   # Ali <= و في حالة إرسال قيمة سيقوم بطباعتها 

# مثال على دالة تعطي مربع القيمة المُدخلة
def square(x) :
   return x * x 

print(square(5))   # => 25

11. التعامل مع الملفات (قراءة وكتابة)

تتيح لك لغة بايثون التعامل مع الملفات المخزنة على القرص الصلب، مما يسمح بحفظ البيانات بشكل دائم واسترجاعها لاحقاً عند الحاجة : 

# الكتابة داخل ملف نصي جديد 
# ( تعني الكتابة وتستبدل المحتوى السابق للملف إن وجد w) 
# encoding="utf-8" إلزامي لدعم الحروف واللغة العربية بشكل سليم استخدام 
with open("notes.txt", "w", encoding="utf-8") as f:
    f.write("مرحباً بك في عالم بايثون !\n")
    f.write(" notes.txt هذا السطر تم حفظه في الملف .\n")

    # القراءة من الملف النصي وعرض المحتويات على الشاشة (r تعني القراءة)
with open("notes.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
💡 نصيحة احترافية: استخدام جملة ()with open هو الخيار الأفضل والآمن دائماً لأن بايثون تقوم بإغلاق الملف تلقائياً فور انتهاء عمليات القراءة و الكتابة، مما يحمي الملفات من التعطل أو الضياع المفاجئ.

12. البرمجة الكائنية (OOP)

البرمجة كائنية التوجه (Object-Oriented Programming): أسلوب برمجي متقدم يهدف لتنظيم الكود وجعله يشبه العالم الحقيقي، عبر بناء قالب عام يسمى الفئة (Class) ونشتق منه مجسمات تسمى الكائنات (Objects).

# (car) باسم (class) تعريف فئة جديدة 
class Car :
    # دالة البناء الأساسية التي تُنفذ فوراً عند إنشاء كائن جديد
    def __init__(self, brand, speed):
        self.brand = brand       # خاصية نوع السيارة
        self.speed = speed       # خاصية السرعة

    # دالة تابعة للفئة تستعرض مواصفات السيارة
    def describe(self):
        print(f"هذه السيارة من نوع {self.brand} وسرعتها القصوى هي {self.speed} كم/س.")

# إنشاء كائنات فرعية ومستقلة تماماً من الفئة الأم
car1 = Car("BMW", 240)
car2 = Car("Toyota", 180)

car1.describe() # يطبع مواصفات السيارة الأولى
car2.describe() # يطبع مواصفات السيارة الثانية
💡
1.class : هو بمثابة قالب او تصميم يتم استخدامه في انشاء الكائنات , في المثال السابق (car).
2.object : نسخة تصف الواقع تم انشاؤها من ال (class) ولكل منها بياناته الخاصة.
3.الدالة (_init_) : تُسمى (constructor) , وهي دالة خاصة يتم استدعاؤها تلقائياً عند انشاء كائن جديد.
4.attributes : بيانات الكائن مثل (brand / speed).
5.methods : دالة تنتمي للكائن و تنفذ عملية ما , مثال : ( ()describe ).

13. الأخطاء والاستثناءات (try & except)

يحمي نظام معالجة الأخطاء البرامج من الانهيار والإغلاق المفاجئ عند قيام المستخدم بإدخال بيانات خاطئة أو غير منطقية :

try:
    num1 = int(input("أدخل الرقم الأول: "))
    num2 = int(input("أدخل الرقم الثاني: "))
    result = num1 / num2
    print(f"نتيجة القسمة الصحيحة هي: {result}")
except ZeroDivisionError:
    print("خطأ: لا يمكن القسمة على صفر رياضياً!")
except NameError:
    print("خظأ!: استخدام متغير لم يتم تعريفه")
except ValueError:
    print("خطأ: يرجى إدخال أرقام فقط وليس حروف!")
except Exception as e :
    print(f"وقع خطأ غير متوقع: {e}")
finally:
    print("هذه الجملة تُنفذ دائماً في نهاية البرنامج سواء نجح أو فشل.")
